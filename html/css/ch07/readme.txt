----------------------------------
1. flexbox에 대해서 알아보자
2. grid에 대해서 알아보자
----------------------------------

rule1
>> flexbox 자식을 원하는 대로 움직이게 하기 위해서는
   기본적으로 부모요소에 명시를 한다.
>> 즉, flexbox-container 로 만들어 주면 된다.

rule2
>> 주축(main axis), 교차축이 있다(cross axis)

flex란? 
: 유연한, 2차원 레이아웃에서 잘 작동한다.
주축(main axis) >> justify-content ( 현재 방향에 주축을 담당하는 속성 )
교차축 >> align-items ( 교차축을 담당하는 속성이다. )

기본적으로 주축은 수평, 교차축은 수직이다.
하지만 default이기 때문에 변경 될 수 있다.
