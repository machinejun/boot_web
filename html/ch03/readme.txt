headtag

- head 태그 안에서 사용하는 모든 태그들은 보여지지 않는다 ( 타이틀 태그 제외 )
- 그러나 글자를 적으면 랜더링 된다 (오류) -> 화면상에 보여질 부분은 body 영역안에 작성해야 한다.
즉, head 태그는 브라우저가 렌더링할 때 도움을 주는 부분이라고 생각하자
